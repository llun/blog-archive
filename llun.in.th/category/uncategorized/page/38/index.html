OpenID ตอนที่ 2

By llun on April 25, 2007 in ของเล่น, มั่วๆ

หลังจากเขียนตอนที่แล้วไป ก็เจอคำถามกลับมาทันทีจาก ที ว่าแล้วมันทำงานยังไง ก็นึกไปซักพักหาของเปรียบเทียบ แล้วก็นึกถึงบัตรเครดิต ขึ้นมาแล้วยกตัวอย่างไป แต่จริงๆ แล้วถ้าจะให้ชัดกว่านั้นน่าจะเป็นพวกสำนักทะเบียนของมหาวิทยาลัย หรือ บัตรเอทีเอ็ม อะไรประมาณนี้มากกว่า มาดูขั้นตอนการทำงานมันดีกว่า (จริงๆแล้วผมก็ยังไม่ได้อ่าน Spec มันละเอียดนะ แค่เอามาใช้จริงเลยเฉยๆ ที่เราก็เลยเหมือนเขียนจากประสบการณ์ อาจจะไม่ตรงบ้าง ไว้อ่านแล้วจะมาเขียนใหม่อีกรอบละกัน) เริ่มจากเราต้องไปสมัครกับผู้ให้บริการ OpenID ต่างๆซะก่อน ก็เหมือนเราสมัครบัตรเอทีเอ็ม ก็จะมีการเก็บข้อมูลส่วนตัวของเราไว้ว่า เราเป็นใคร และรหัสไว้สำหรับยืนยันตัวเอง อะไรประมาณนี้ จากนั้นเราก็จะได้ uri มาซึ่งก็เหมือนพวกบัตรเอทีเอ็ม เมื่อเราต้องการใช้บริการต่างๆ ที่รองรับ OpenID เราก็เอา uri ของเรากรอกเข้าไป จากนั้นตัวบริการที่เราต้องการใช้ก็จะไปถามว่า เครื่องที่เราใช้เนี๊ยะ คือคนๆนี้จริงๆ หรือป่าว โดยการส่ง uri ของเว็บและ uri ที่เราอกรอกไปยัง server ของ OpenID ใน uri ของเรา จากนั้นถ้า server OpenID [...]

หวัด

By llun on April 23, 2007 in มั่วๆ

เมื่ออาทิตย์ที่แล้วเจ็บคอ ก็สั่งยามากิน กินหมดตอนแรกนึกว่าหายแล้วที่ไหนได้ มาอาทิตย์นี้ หนักกว่าเดิมอีก ทั้งน้ำมูก ทั้งไอเลยต้องหายามากินเพิ่มแย่จริง สงสัยวันไหนต้องไปหาหมอซะหน่อยแล้ว หาแต่เภสัชหน้าหมู่บ้านท่าทางจะเอาไม่อยู่ ตอนนี้ไปทำงานทีก็แทบแย่แหนะ เลยหาทางแก้หาที่นั่งที่มีที่พิงแล้วก็หลับตลอดทางซะเลย(มีบ้านกับที่ทำงานอยู่ต้นสาย ปลายสายมันดีอย่างนี้นี่เอง ไม่ต้องกลัวเลย ) แต่พอตื่นมางัวเงีย โอย ไม่อยากตื่นเลย งืดๆ วันนี้ทำงานก็แทบแย่ ทำไปจามไป แอร์เสียอีก สรุปวันนี้แย่ทั้งวัน ปอลอ. เมื่อไหร่จะฝนตกฟะ ร้อนอิ๊บอ๋าย ตากแดดทีป่วยเลย ปอลอสอง. ใครชวนไปกลางแจ้งจะปฏิเสธทันที ขอนอนอยู่บ้านดีกว่าหละคร๊าบ

WordPress + Google calendar

By llun on April 17, 2007 in มั่วๆ

ผมนั่งหา Plugin มาตั้งนานแล้วแต่หาที่ถูกใจเกี่ยวกับ Calendar ใน WordPress ไม่ได้เลย โดย Calendar ที่ผมต้องการนั้น ต้องการเอามาแทนที่อันเก่า ที่แสดงสีเฉพาะวันที่มีการ post เท่านั้น หา plugin ส่วนใหญ่ก็จะแสดงเป็น agenda (แสดงเป็นรายการ ไม่ได้เป็นรูป Calendar) หรือก็ต้อง post ใน wordpress อีก สุดท้ายก็เลยคิดจะทำเองซะเลย Requirement ของผมนั้นง่ายมาก เอา ics จากที่อื่นมา render เป็นรูป Calendar โดยเน้นสีต่างกันในแต่ละเหตุการณ์และในแต่ละวัน (วันไหนมีหลายเหตุการณ์ ก็แสดงสีที่ต่างไปอีกสีหนึ่งเลย ไม่ได้คิดจะทำสีรุ้งหรอกนะ ) อืมฟังเหมือนง่ายแฮะ แต่ยังไม่ได้ลองทำเลย(สงกรานต์มัว เที่ยว นอน ดูวิดีโอ เปลี่ยนสีเว็บ ก็หมดเวลาซะและ ) คาดว่าจะเริ่มเสาร์อาทิตย์นี้ ได้ยังไงเดี๊ยวก็มาดูกัน

เที่ยวตลาดน้ำ

By llun on April 15, 2007 in มั่วๆ

สงกรานต์ปีนี้กะว่าจะนอนอยู่บ้าน ดูหนังซะหน่อย สุดท้ายแล้ว เมื่อวานก็โดนแม่ลากไปตลาดน้ำอัมพวาซะนี่ แต่ว่าไม่ได้ไปแค่ตลาดแต่ได้ล่องเรือตามแม่น้ำท่าจีนด้วยนี่สิ จริงๆ แล้วการเดินทางที่ชอบที่สุดนี่ก็เป็นทางเรือนี่แหละ ทุกวันนี้ก็เดินทางด้วยเรือเกือบทุกวัน เพราะว่า ไม่มีการจราจรติดขัดให้รู้สึกอึดอัดขณะเดินทาง + อากาศที่เย็นกว่าอยู่บนถนน (ถ้าไม่นั่งรถแอร์) งานนี้ก็เลยยอมไปแต่โดยดี และก็เอาภาพต่างๆ เก็บไว้ด้วยตาม Link ด้านล่างเลย สำหรับใครที่ชวนเที่ยวครั้งต่อไป ขอน้ำตก ขึ้นเขา ล่องแม่น้ำอย่างนี้นะ อย่าชวนไปทะเล หาดทรายเลย เบื่อแล้ว
