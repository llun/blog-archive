KMS

By llun on March 26, 2010 in ของเล่น, ระบบ

อาบน้ำอยู่แล้วมันแวบเข้ามาในหัวแล้วคิดยาวมาเรื่อย ๆ เลยเขียนซะเลย แถมอาทิตย์นี้ทั้งอาทิตย์ปั้นระบบ KMS ของบริษัท แล้ววันนี้ได้อะไรมาพอสมควร เขียนเป็นข้อ ๆ เลยละกัน​ (ไม่เรียงลำดับและความต่อเนื่องนะ)

  • KMS จริง ๆ แล้วไม่จำเป็นต้องเป็นระบบอะไรที่ยิ่งใหญ่ ราคาแพง แค่ WordPress ก็เพียงพอแล้ว แต่ต้องมีช่องทางให้ทุกคนใส่ข้อมูลไปได้ง่าย ๆ และระบบค้นหาที่ดีก็พอ
  • จะให้ทุกคนเข้ามาถ่ายทอดความรู้ได้ กฏการใส่ข้อมูลต้องน้อยที่สุด ไม่จำเป็นต้องเป็นภาษาอังกฤษเท่านั้น และไม่จำเป็นที่จะใส่ข้อความยาว ๆ จะใส่สั้น ๆ ก็ได้ แต่อย่างน้อยต้องเลือกประเภท และใส่ Tag ที่เกี่ยวข้องเพื่อให้หาได้ง่าย ๆ
  • Category คือหมวดหมู่ใหญ่ ๆ ที่มีลำดับชั้นไม่มาก ไม่ควรเกินสาม (จริง ๆ แค่สองก็รู้สึกเยอะแล้ว) ส่วน Tag คือส่วนที่ไว้แบ่ง Category แบบละเอียด และทำให้หมวดหมู่มันปนกันได้
  • Visual Editor จำเป็นมาก อย่าคิดเอามันออกเด็ดขาด ใช่ว่าทุกคนจะชอบพิมพ์ HTML หรือ Markdown สด ๆ
  • ถ้าใช้ WordPress ควรหา plugins ไว้ Embed Video มันช่วยได้เยอะมาก
  • WordPress สามารถติด plugins เพื่อกันคนภายนอกเข้ามาแอบอ่านได้ แต่ มันไม่กัน Feed ให้เพราะงั้น ถ้าเอาไว้ที่โฮสท์ภายนอก ใส่ options กันไว้ใน Apache ด้วย! ใส่ประมาณ code ด้านล่างนี้แหละ 
        <Location ~ "(\w|\/)*/feed/?$">
      Order deny,allow
      Deny from all
    </Location>
  • การเขียนเรื่องใหญ่ ๆ ควรแบ่งย่อย และค่อย ๆ ใส่อย่าเอาเรื่องเดียวกันใส่วันเดียวหมด ถ้าอยากให้คนอ่านง่าย ๆ และสนใจ (มีพี่ใส่วิธีใช้ command line unix เกือบทั้งหมดที่ใช้ ใน post เดียว หาคำสั่งกันเหนื่อยเลยใน post นั้น คนใส่เนื้อหาง่าย แต่คนอ่านลำบาก)
  • เอกสารที่ไม่สามารถค้นหาได้ ควรเก็บแยกไว้ หรือหาอีกระบบมาช่วยทำให้มันค้นหาได้ เพราะไม่งั้น post นั้นจะหายไปจากความจำอย่างรวดเร็วและดองไว้จนหาไม่เจอ (ถ้าไม่มีข้อความใดแปะไว้เพื่อให้หาง่าย ๆ เช่น Tag, Category และ หัวข้อ)
  • พี่ที่ทำงานมานานแล้ว ไม่ค่อยเข้ามาเขียนอะไรพวกนี้ ส่วนมากจะทำเป็นเอกสารด้วย Word หรือ Powerpoint เลยมากกว่า หรือไม่ก็จะใช้วิธีเล่าเอา หรือให้ไปหาใน Google ยังหาวิธีจัดการกับ ความรู้ พวกนี้ไม่ได้ (จริง ๆ ที่บ้านเคยคะยั้นคะยอให้พ่อเขียน ก็ไม่ได้ผลเหมือนกัน คงเหมือนที่ @teerapapc และ @visibletrap บอกไว้ ว่าถ้าคนอยากเขียน ก็จะเขียนไปบังคับไม่ได้ แต่คนอยากรู้อย่างผม อยากได้สิ่งที่อยู่ในนั้นแบบไม่ต้องการให้ปล่อยผ่านนี่สิ) เลวร้ายสุดอาจหาคนถามสดแล้วจด

    • อาทิตย์แรกก็ได้มาประมาณนี้แหละ ตอนนี้พยายามหาของเล่นยัดเข้าระบบนี้จนมันจะเป็นเหมือน Portal มากกว่าแล้ว (มีทั้ง Twitter List, Calendar, Bookmarks) ส่วนอื่น ๆ ไว้คราวหน้าละกัน จริง ๆ อยากเรียนโทต่อด้านนี้นะ แต่ไม่รู้ต้องเข้าคณะอะไรหรือที่ไหน หรือมันเรียกว่าอะไร (มองไปที่ MIS มหาวิทยาลัยต่าง ๆ เหมือนเรียนบริหาร เลยไม่ค่อยอยากเรียน) สงสัยต้องลองผิดลองถูก หาหนังสืออ่านเองไปเรื่อย ๆ ก็หวังว่าจะได้ KMS ที่ถูกใจซักวัน
