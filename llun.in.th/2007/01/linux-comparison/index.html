Linux comparison

By llun on January 12, 2007 in ระบบ

วันนี้เกิดอาการง่วงเหงาหาวนอนตั้งแต่เช้า เนื่องด้วยเมื่อวานนอนไม่หลับ (นอนคิดทั้งคืนเลยว่าจะทำงานที่ Throughwave ดีไหม สุดท้าย ก็ไปทำงานที่นั่นหละครับ ) Project ก็เลยไม่เดินไปด้วยเลย ไว้พรุ่งนี้ค่อยทำต่อละกัน ยังไงก็อยู่ภายในอาทิตย์นี้แหละ เห้อมาพูดถึงเรื่อง Linux ดีกว่า หัวข้อที่เขียนคราวนี้ก็ไม่ไช่อะไรหรอกอยากจะสับ Ubuntu ต่อ เพราะนอกจากความง่ายของมันแล้ว อย่างอื่นรู้สึกมันติดขัดไปหมดยังไงไม่รู้

Ubuntu 6.10 openSUSE 10.2 Solaris 10
Installation โคตรง่ายเลย แถมติดตั้งไปเล่นเนทไปได้ด้วย นี่แหละข้อดีอย่างนึงของ Ubuntu ขั้นตอนเหมือน Linux ทั่วไป คือเลือก Package ก่อนแล้วก็แสดงว่าไปถึงไหนแล้ว แต่นานมากกกก ระบบติดตั้งของ solaris เป็นอะไรที่ห่วยมากเลยครับพี่น้อง คือจะเป็น text ก็ไม่ไช่แต่เป็น ui ที่ทำออกมาแปลกๆ อ่ะ ไม่ประทับใจเอาซะเลย
Application มีมาให้เลยโดยไม่ต้องเลือก ก็ถือว่าดีสำหรับคนที่ใช้อะไรพื้นฐาน สามารถเลือกก่อนไ ด้ว่าจะลงอะไร ไม่ลงอะไร เนื่องจาก application สำนักงานที่ดังๆ คือ openoffice ทำมาจาก sun มันก็เลยมี application ที่ติดตั้งง่ายเยอะแยะไปหมด ชอบจริงๆ เลย
Internationalize(Thai) ดีมากๆๆๆๆๆ มี package ให้พร้อม ชอบที่ซู๊ดเลย ต้องหา package มาติดเพิ่มเอง …ถือว่า OK ครับไม่ต้องติดเพิ่มแต่คงต้องเพิ่ม font ซะหน่อย
Support มีแหล่งปรึกษาที่เยอะมาก ทั้งไทยและเทศ (ใครเล่นเข้า Ubuntuclub.com ด้วยนะครับ ) มีแต่เว็บฝรั่งแถมข้อมูลน้อยกว่า Ubuntu มากกกกก แต่ก็ยังโอเค…
Stable(Application not OS) โคตรน้อย เปิด อะไรนิดหน่อยไปแล้ว เซ็งมากๆ โดยเฉพาะ Evolution เยี่ยมโคตรๆ ผมแทบไม่ต้อง force ปิดโปรแกรมไรเลย ข้อนี้แหละที่ openSUSE เอาใจผมไปเต็มๆ ถือว่าโอเค แต่ก็ยังชอบ openSUSE มากกว่าอยู่ดี

จริงๆ ยังมีอะไรอีกเยอะที่อยากเพิ่ม แต่ยังนึกไม่ออก ไว้มาเพิ่มทีหลังละกัน
