Social Network Service

By llun on September 22, 2007 in ของเล่น

เมื่อวานเพื่อนชวนให้อัพโหลดรูปลง multiply โดยการให้ account multiply ที่สร้างขึ้นมาใหม่แล้วแจกให้กับรุ่น ดังนั้นใครก็สามารถเข้าไปแก้ไขข้อมูลอะไรก็ได้ รู้สึกมันไม่ปลอดภัยยังไงชอบกล แต่ถ้าดูเจ้าอื่น วิธีการทำก็น่าจะคล้ายๆ กัน เท่าที่เห็นแล้วแต่ต่าง จากกลุ่มก็คงมีอยู่เจ้าเดียวตอนนี้คือ facebook facebook แหวกแนวจากเจ้าอื่นตรงที่แทนที่จะทำไรหมดทุกอย่าง(จริงๆ ก็ทำเหมือนกัน) แต่กลับมี api ให้คนอื่นเขียนโปรแกรมแล้วมาต่อกับ facebook ได้ ผมเลยไม่จำเป็นต้องย้ายรูปจาก picasaweb มา facebook เองด้วยมือ หรือจะเอา event จาก google calendar มาใส่ก็ได้ โดยที่ไม่ต้องย้ายมาสร้างใหม่จาก facebook ถ้าเป็นที่อื่นคงต้องหาทาง import มาเอง แต่จะให้คนอื่นเปลี่ยนมาใช้ facebook เหมือนกันคงลำบากแล้วหละ เพราะคนอื่นใช้ multiply และอัพโหลดรูปเขียนอะไรไว้เยอะแยะหมดแล้ว ก็หวังว่าจะมี app ใน facebook ที่เชื่อม service ของเจ้าอื่นบ้างนะ (นี่สินะที่เรียกว่า switching cost งืดๆ)
