ย้าย Server (อีก)แล้ว

By llun on April 6, 2008 in มั่วๆ

คราวนี้คงอยู่แบบถาวรแล้วหละครับ แหะๆ เนื่องจากเป็นโฮสท์ส่วนตัวและหลังจากย้ายหลายรอบเหลือเกิน (รวมถึงครั้งล่าสุดก็สี่รอบหละ) แต่ตอนนี้ตัว server ยังไม่เสร็จดี ขาดอีกหลายๆ อย่างให้โค้ดช่วยทำอยู่ เช่น mirror และ jail อีกอย่าง control panel ก็ยังไม่มี กำลังคิดอยู่ว่าจะเขียนเอง แต่ไม่รู้จะไหวป่าว เดี๊ยวดูช่วงสงกรานต์นี้แหละครับ

อีกอย่างที่ยังขาดไปคือ trac ที่มี svn แล้วแต่ยังไม่ได้ทำอะไรต่อเลย มัวแต่ยุ่งๆ กับงานประจำอยู่อีกนั่นแหละ เอาเป็นว่า ทุกอย่างคงลงตัวภายในเดือนนี้แหละครับ เดี๊ยววันนี้เอา trac ให้ขึ้นแบบไม่มี svn อย่างน้อยจะได้ใส่ task ที่ต้องทำกับ server ลงไป ฮะๆๆ โอยเหนื่อย

ปอลอ. แบก server ข้ามสะพานลอยนี่เป็นจุดเด่นดีเหมือนกันแฮะ แถมได้กล้ามเพิ่มมาอีกต่างหาก T T”
ปอลอสอง. mirror แบบชั่วคราว http://mirror.llun.info
ปอลอสาม. ยังคิดชื่อโดเมนใหม่สำหรับโฮสท์โดยเฉพาะไม่ออกเลยงะ
