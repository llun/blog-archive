ลาก่อนชีวิตในมหา'ลัย

By llun on January 27, 2007 in มั่วๆ

เมื่อวานไปงานบายเนียร์มา(เลยมา 30 นาทีแล้วก็เรียกว่าเมื่อวานเลยละกัน ^^!) เป็นงานที่อิ่มมากๆ เอ้ย สนุกมากๆ (หรือป่าว) ถ่ายรูปกับคนอื่นถ้วนหน้าทั้งๆ ที่ไม่มีกล้องตัวเองเลย TT” ยังไงใครถ่ายไป ฝากส่งมาด้วยนะคร๊าบบบ งาน นี้ก็เป็นงานที่เลี้ยงสำหรับอำลารุ่นพี่ๆ ที่กำลังจะจบ จากมหา’ลัย อันแสนสบาย เพื่อมาทำงาน หรือเรียนต่อ ซึ่งมองไปแล้วอยากให้การทำงานเหมือนเรียนในมหา’ลัยซะจริงๆ กำหนดเวลาไปเรียนเองได้ สามารถแก้ไขงานต่างๆ ได้โดยไม่โดนเจ้านายว่า เห้อ ก็หวังว่างานที่เลือกมาคงจะเหมาะกับเรานะ เพราะไม่อยากเป็นมนุษย์เงินเดือนเหมือนคนอื่นเลยจริงๆ และอีกอย่างไม่อยากลืมความฝันหรือเป้าหมายที่อยากทำด้วยนี่สิ ว่าแล้วไปอัพบล๊อกนู้นหน่อยดีกว่า <<” ปอลอหนึ่ง. วันนี้ในงานมีวิดีโอตอนปีหนึ่งที่ถ่ายไว้ตอนงาน com-sci night ดูแล้วก็ฮาดี ตอนนั้นเต้นไปได้ยังไงนะ สงสัยเจงๆ ปอลอสอง. เพื่อนๆ ยังไม่ได้ปรับเวลามาทำงานตอนเช้าเลย สงสัยเราต้องจูนเวลามาทำงานตอนกลางคืนก่อนมั่งดีไหมเนี๊ยะ ปอลอสุดท้าย. อยากได้ Mac Book Pro (เกี่ยวกับหัวข้อตรงไหนหว่า 555+)

StringBuffer กับ String

By llun on January 21, 2007 in ภาษา

คำถามนี้ถูกถามตอนที่ Througwave มาสัมภาษณ์ ว่า StringBuffer นั้นดีกว่า String ยังไงให้อธิบาย ก็อึ้งไปซักพักพร้อมตอบไปว่าไม่รู้ วันนี้ว่างๆ (จริงๆ ก็ไม่ว่างหรอก แต่ขี้เกียจ) เลยมานั่งหาคำตอบซักหน่อย แล้วก็ได้เจอคำอธิบายจากเว็บนี้ http://www.javaworld.com/javaworld/jw-03-2000/jw-0324-javaperf.html ดูจากเว็บแล้วก็เห็นกันจะๆ เลยว่า จริงๆแล้วเวลาจะทำอะไรเกี่ยวกับ String จะมีการสร้าง StringBuffer ทุกครั้งในตัวอย่างเค้ายกการต่อ String ให้ดู เพราะฉนั้นต่อไปหากจะทำอะไรเกี่ยวกับ string ใช้ StringBuffer ดีกว่าแล้วสุดท้ายค่อยมาแปลงเป็น String ตอนจบหละครับ ปล. จริงๆ มีอีกคำถามแต่ช่างมันก่อน
