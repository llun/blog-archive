Live CD

By llun on June 16, 2007 in ระบบ

หลังจากที่ Ubuntu ออกมาแบบ Live CD สามารถติดตั้งได้โดยใช้แค่ CD แผ่นเดียวจนคนส่วนใหญ่ติดใจ Linux ค่ายอื่นๆ ก็เริ่มทำตามกันบ้านแล้วแฮะ เริ่มจาก Fedora ก่อนเลยที่ Version7 ออกมามีแบบ Live CD ให้โหลด แถมแบ่งเป็นสองแบบคล้ายๆ Kubuntu กับ Ubuntu อีกต่างหาก สองวันก่อนใน Distrowatch ก็ไปเห็น openSUSE alpha 5 ก็เอาบ้างและ แม๊ กระแสนี้มาแรงจริงๆ จะว่าไปผมก็ชอบนะ Live CD มันทำให้เห็นก่อนว่าระบบที่เราจะติดตั้งเป็นยังไง แต่ที่ผมอยากได้จริงๆอ่ะ คือ Live แล้วมีให้เลือกด้วยได้ป่าวว่าจะติดอะไรลงไปมั่ง อย่าง เกม กับ Evolution(โปรแกรมอ่านอีเมล์) อย่างงี้อ่ะ ไม่อยากติดจะได้เลือกออกได้ (จะได้ไปติด Thunderbird ทีหลังแทน) เอ หรือจะมีใครทำ Live DVD ออกมาบ้างหรือป่าวนะ
